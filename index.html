Description of WebPalm

WebPalm is a large-scale public dataset for contactless palmprint recognition. This dataset allows us to assess the performance of palmprint recognition models with a minimal False Acceptance Rate (FAR) of 1e-9.

WebPalm consists of 83,145 palmprint images of 83,145 unique identities, collected from public internet resources. The dataset encompasses different acquisition devices, varied lighting conditions, and a range of user poses, therefore allowing for a realistic simulation of large-scale scenarios.
